# webflix

![](./.gitlab/logo.png)

## Build

### **webflix.js** (backend)

```
~$ make library  # ./build/webflix.js
```

### Mozilla Firefox extension

```
~$ make firefox  # ./build/firefox
```

### Google Chrome extension

```
~$ make chrome  # ./build/chrome
```
